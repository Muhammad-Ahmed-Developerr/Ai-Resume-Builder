# AI Resume Builder

Create professional, ATS-friendly resumes in minutes with an intuitive AI-powered resume builder.

Built using **Next.js**, **TypeScript**, and **Tailwind CSS**, this application allows users to enter their personal information, education, work experience, projects, certifications, and skills, then generate beautiful resumes from multiple modern templates and export them as high-quality PDF files.

## Installation

Clone the repository:

```bash
git clone https://github.com/Muhammad-Ahmed-Developerr/Ai-Resume-Builder.git
```

Navigate to the project folder:

```bash
cd ai-resume-builder
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser:

```text
http://localhost:3000
```

## How It Works

1. Enter your personal information.
2. Add your education details.
3. Add work experience.
4. Include skills and technologies.
5. Add projects and certifications.
6. Choose a resume template.
7. Preview your resume instantly.
8. Download your resume as a PDF.
